# Review-Facing Result Tables

This directory contains compact result tables included for manuscript peer review. Raw experimental workbooks and full generated analysis folders are intentionally excluded from the public repository.

## Included Tables

- `peer_review/main_target_internal_external_metrics.csv`  
  Internal formulation-level cross-validation and external-validation metrics for the main rheological targets: viscosity, apparent yield/breaking stress, breaking strain, `G'` at 1 Hz, and `eta50`.

- `peer_review/all_model_metrics_internal_external.csv`  
  Model-suite comparison across formulation-only, formulation plus `G'` at 1 Hz, formulation plus `eta50`, and combined feature sets.

- `peer_review/surrogate_validation_metrics.csv`  
  Surrogate-model validation metrics used to support inverse-design scoring.

- `peer_review/recommended_experimental_validation_candidates.csv`  
  Final inverse-designed formulations selected for experimental validation, with target values, measured values, and relative errors.

## Notes

The tables are generated by scripts in `scripts/` and are provided to allow reviewers to inspect the numerical basis of the figures and manuscript claims. They should be regenerated from the full raw-data package when the final data deposition is prepared.
