# Review-Facing Figures

This directory contains selected publication-ready PDF figures for peer review.

## Included Figures

- `Figure_main_targets_R2_internal_external_large_labels_600dpi.pdf`  
  Internal cross-validation and external-validation `R^2` for the main rheological prediction targets.

- `Figure_main_targets_RMSE_internal_external_large_labels_600dpi.pdf`  
  Internal cross-validation and external-validation RMSE for the main rheological prediction targets.

- `Figure_laos_combined_R2_RMSE_by_feature_combination_panel_AB_600dpi.pdf`  
  Combined `R^2` and RMSE comparison across feature sets for LAOS-derived targets.

- `Figure_inverse_design_target_library_large_labels_600dpi.pdf`  
  Inverse-design score maps for target viscosity and elastic-modulus windows.

TIFF exports and full generated output folders are excluded to keep the public review repository lightweight.
